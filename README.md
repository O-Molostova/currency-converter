# Currency Converter

A simple currency converter built with React, TypeScript, and Axios using the CurrencyBeacon API.

## Features
- Select "From" and "To" currencies from the list provided by the API.
- Enter an amount to convert.
- Automatic conversion when amount or currencies change.

## Tech Stack
- React 18 + Vite
- TypeScript
- Axios

## Getting Started

The app is deployed on Netlify for easy UI and functionality testing: https://currency-converter-olena.netlify.app/
